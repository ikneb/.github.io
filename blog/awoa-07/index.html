A Week of A-Frame 7

Mar 4, 2016

What's up with A-Frame for the week of Feb 26, 2016 to Mar 4, 2016.

A-Frame is a WebVR framework for building virtual reality experiences with HTML.

Projects

  • swamphousegroup built WebVR Forest —

    At A-Frame National Park.

  • dkraeker built iSpyStore —

    E-Commerce spy gadget store.

  • jon_ibe built Breathing Scape —

    A tree-adorned hill that slowly breathes.

  • chris_muetze built Hello Virtual World —

    Moody dark scene with cool shadows, reflections, and animated clouds.

  • alexoviedo999 built Tron World —

    Jump through a wormhole into the world of Tron.

  • joshgalvin built A-Runner —

    Infinite runner game with cubes.

  • joshgalvin built FireFrame —

    Multiuser experiment using Firebase for synchronizing users connected to the same A-Frame scene.
