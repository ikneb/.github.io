A Week of A-Frame 4

Feb 12, 2016

What's up with A-Frame for the week of Feb 5, 2016 to Feb 12, 2016.

A-Frame is a WebVR framework for building virtual reality experiences with HTML.

Projects

  • donmccurdy built Monument Valley —

    A scene from Monument Valley. Exported using ephtracy’s MagicaVoxel. Hooked up to custom components for jump controls, USB gamepad controls, remote mobile controls using WebRTC, and customized keyboard controls.

  • meta-meta built Aztec Pong —

    Treat yourself to a good ol’ fashioned game of pong in a lush Aztec-esque terrain.

  • ngokevin built Minecraft Viewer Boilerplate —

    A boilerplate for viewing Mineways-exported Minecraft scenes.
